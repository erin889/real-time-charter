# Real Time Charter

## Quick Start

Clone this repository:

```bash
$ cd my-projects-env
$ git clone https://github.com/erin889/real-time-charter.git
```

Spining up the application:

```bash
cd real-time-charter
rails s
```

Go to [http://localhost:8000](http://localhost:3000/music-charts).
